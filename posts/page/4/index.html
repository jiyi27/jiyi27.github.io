1. e.preventDefault()

function sendMessage(e) {
    e.preventDefault()  // Submitting form would refresh the page, so we prevent page reload
    const input = document.querySelector('input')
    if (input.value) {
        ws.send(input.value)
        input.value = ""
    }
    input.focus()  // Focus the input after sending the message
}

document.querySelector('form').addEventListener('submit', sendMessage)

Usually used to prevent page reload when submit form.

#c# 

1. Data Source

By default, model binding gets data in the form of key-value pairs from the following sources in an HTTP request:

  1. Form fields (case insensitive)
  2. The request body (For [controllers that have the ApiController] attribute.)
  3. Route data
  4. Query string parameters
  5. Uploaded files

2. Simple types

Model Binding in ASP.NET Core | Microsoft Learn

3. Complex Type

A complex type must have a public default constructor and public writable properties to bind. When model binding occurs, the class is instantiated using the public default constructor.
