什么是一个好的论证

2025-02-011 分钟阅读 (140 字数)
#读书  #逻辑 

1. 严谨的结构#

决定什么是良好的论证，第一项标准，是看它是否拥有坚牢的逻辑结构。它的结论必须由至少一个前提支持着，它的结构形式则必然是下面两种情况之一，如果是演绎性的，结论必然地从前提导出，如果是归纳性的，结论或然地从前提导出。另外，如果是规范性论证，必须有一个规范性的前提。

这样的论证，不使用相互冲突或与结论相矛盾的推理，也不预先假设结论的真实性，更不进行不合逻辑的演绎推导。演绎逻辑的定律已经建立得相当完好，不论是在假言推理还是三段论推理中。违反其它的任何一种规则，都只会带来演绎性论证的结构缺陷。

论证可以按照推理方式分类: 演绎和归纳, 也就是说一个论证要么是演绎性的, 要么就是归纳性的.

论证也可以分为事实和规范论证,

演绎推理又可以分为常见的两种: 假言推理和三段论推理

注意, 道德前提, 法律前提, 审美前提, 价值观前提这些东西往往在一个论证中属于隐含前提, 挖掘出规范性论证中的这些隐藏前提, 可以清楚发现双方的意见的分歧, 辩论更有效率,

判断论证是否良好的第二条原则，与前提的相关性有关。好论证的前提，一定与结论的价值或真实性有关，如果一个前提与结论的真实与否两不相涉，那么，有什么理由去花时间考虑前提是不是真实、能不能够接受呢？

判断论证是否良好的第三项规范，与前提的可接受性有关。要支持一个结论，那理由必须是可以接受的。理由是可以接受的，意思是说，对理智的人来说，在所有能够获得的相关证明面前，当能接受那声称。

一个论证，还必须满足良好论证的第四项标准的要求，那便是充分原则。论证必须有数量和质量都充足的相关且可接受的前提，才能使我们觉得这论证足够有力，可以接受它的结论了。

例子:

论述: 你认为西方民主很好嘛?马斯克明目张胆地为特拉票，用钱开道，所谓民主国家的通病，民主选举选出来希特勒，民主投票让亚里士多德放逐，所谓民主不过是多数人对少数人的暴政，而且换而言之中国没有民主嘛？基层选举你不知道嘛

第一步：检查是否符合良好论证的四项规范#

1. 形式完好的论证结构（第一项规范）#

即，论证不能直接假设结论为真，也不能使用矛盾推理或不合逻辑的推导。

  • 陈述的核心观点：

    • 西方民主并不好，因为：
      1. 马斯克用钱影响选举，民主容易被资本操控。
      2. 民主曾经选出了希特勒，民主投票曾放逐亚里士多德。
      3. 民主是多数人对少数人的暴政。
    • 中国也有民主，因为有基层选举。

  • 分析：

    • 该论证表面上有逻辑结构，但存在多个问题：
      • 过度概括（hasty generalization）：
        • 马斯克的个人行为 ≠ 整个西方民主的本质。
        • 历史上的极端个例（希特勒、亚里士多德） ≠ 现代民主制度的运作方式。
      • 隐含假设（implicit assumption）：
        • “民主选出了希特勒” → 但现代民主制度有更完善的制衡机制，这个例子是否仍然适用？
        • “民主是多数人对少数人的暴政” → 但许多民主国家有宪法保护少数群体，是否完全是“暴政”？
        • “中国有基层选举，所以有民主” → 但民主不仅仅是选举，是否能代表整个民主体系？
      • 偷换概念（equivocation fallacy）：
        • 民主的定义是多层面的，仅仅用“基层选举”来定义民主，可能过于狭隘。

❌ 该论证的结构存在推理漏洞，不能完全符合第一项规范。

2. 提出的理由是否真正支持结论（第二项规范）#

即，提出的论据是否对结论的真实性有所贡献，而不是无关或弱相关的内容。

  • 检验各个论据与结论的关系：
    • 马斯克用钱影响选举 → 西方民主不好
      • 问题：资本对民主的影响确实存在，但这并不能证明“西方民主整体上不好”。
      • 反驳：许多民主国家有竞选资金监管制度，并非所有选举都被金钱操控。
    • 民主选出希特勒、投票放逐亚里士多德 → 证明民主不好
      • 问题：这是以个别历史事件代表整个民主制度，属于以偏概全（cherry picking）。
      • 反驳：现代民主制度已发展出更复杂的机制来防止极端情况（如权力制衡、普世人权）。
    • 民主是多数人对少数人的暴政
      • 问题：民主制度中通常有少数派保护机制，如宪法权利、司法独立，不能简单等同于暴政。
    • 中国有基层选举 → 证明中国有民主
      • 问题：民主不仅仅是选举，还包括权力分立、新闻自由、言论自由等要素，仅仅有选举不能证明整体民主。

❌ 该论证的前提与结论的关系较弱，不完全符合第二项规范。

3. 提出的前提是否可接受（第三项规范）#

即，论证中使用的事实和数据是否是理性的人可以接受的。

  • 哪些前提可接受？
    • “马斯克影响选举” → 部分可接受，确实存在资本对选举的影响，但未必全盘否定民主制度。
    • “民主选出希特勒” → 事实可接受，但逻辑推论有问题，因为纳粹的掌权不仅仅是民主的结果，也涉及经济危机、宣传手段等因素。
    • “民主投票放逐亚里士多德” → 历史上确实发生过，但它代表古希腊民主，而非现代民主。
    • “民主是多数人对少数人的暴政” → 这个说法过于极端，现代民主通常有少数派保护机制。
    • “中国有基层选举” → 事实可接受，但是否能代表完整民主制度存疑。

✅ 部分前提出自事实，但许多推论不够严谨，因此部分符合第三项规范。

4. 论证的数量和质量是否足以支持结论（第四项规范）#

即，论证是否提供了足够的证据支持结论，且质量足够高。

  • 证据数量：
    • 该陈述只举了有限的几个例子（马斯克、希特勒、亚里士多德），但要论证“西方民主不好”或“中国有民主”，需要更系统的数据支持，而不仅仅是几个历史事件。
  • 证据质量：
    • 马斯克的例子：是个别现象，未能代表整体。
    • 希特勒的例子：属于特殊情况，不能完全否定现代民主。
    • 亚里士多德的例子：与现代民主的关系不大。
    • 基层选举的例子：只是民主的一部分，不能代表完整民主体系。

❌ 该论证的证据数量和质量都不足以支持结论，不符合第四项规范。

第二步：运用辩驳原则#

如果仍然无所适从，我们可以从以下角度进一步检验该论证的合理性：

  1. 是否忽视对自身论证的主要疑问？

    • 该陈述忽视了现代民主与历史民主的区别，没有考虑现代民主制度的演进。
    • 该陈述没有回答：“民主是否有制衡机制来防止资本影响选举？”、“民主是否有少数派保护机制？”

  2. 是否无法有效回应主要疑问？

    • 如果问：“民主难道没有优点，比如公民自由、透明政治等？” 该陈述没有回应。
    • 如果问：“资本影响选举是否代表民主整体无效？” 该陈述没有足够数据支持其观点。

  3. 是否忽视对方论证的弱点？

    • 该陈述没有讨论民主制度如何保障少数派权利，而是直接称其为“暴政”。
    • 该陈述没有考虑“资本影响”是否可以通过法律和制度改善。

  4. 是否缺乏能力对此进行盘诘？

    • 该陈述提出了批评，但未能深入探讨民主制度的改进可能性，只停留在批评层面，没有提供替代方案或更好的政治模式。

❌ 在辩驳原则的检验下，该论证存在大量漏洞，难以支持其结论。

最终结论#

经过严密分析，我们发现：

  • 该陈述在逻辑推理上存在多个漏洞，且论据不足，无法支撑其强烈的结论。
  • 虽然个别前提是事实（如资本影响选举、历史上的民主决策失误），但它们并不能直接证明“西方民主不好”或“中国有民主”。
  • 辩驳原则表明，该论证没有充分回应关键问题，缺乏对民主制度完整性的讨论，且忽视了民主制度的进步与制衡机制。

因此，该论证的说服力较弱，不能得出“西方民主不好”或“中国有民主”的结论。
