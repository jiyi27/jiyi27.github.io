1. 创建一个简单的 spring boot 项目 - Controller 类

创建一个 spring 项目思路, 首先要想到创建的一个类是 Controller, 因为我们要处理不同的 endpoints, 比如叫 WebController.java, 名子无所谓, 只要加上 @Controller/@RestController 的类都是 Controller, 然后考虑你的这个 Controller 是要返回简单的 json 对象还是 web 页面, 前者的话就选择 @RestController 修饰你的 Controller, 至于这两个注解的区别, 可以参考: Spring Boot项目及踩坑总结 Spring学习(一)
