dict,

```python
dict = {'a': 1}

# This will raise a KeyError
print(dict['b'])

# Using .get() with a default value 
print(dict.get('b', 0)) # Print 0
```

slice, 

```python
arr = [1, 2, 3, 4]
print(arr[-1])
// 4

for i in range(3):
    print(i)
// 0 1 2
```
